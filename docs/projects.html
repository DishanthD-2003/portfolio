My Projects



Predicting Cyber Attacks Using IDS


This project focuses on detecting potential cyber attacks by analyzing network traffic using machine learning techniques. The system processes network data to identify unusual patterns that may indicate malicious activity. Data preprocessing and feature engineering were performed to improve the accuracy of the detection model. The model was trained to classify normal and suspicious network behaviour effectively. This helps in early detection of security threats and improves overall network protection. The project was developed using Python, machine learning libraries, and SQL for data handling.


Tech Stack: Python, Machine Learning, SQL



Virtual Assistant for Visually Impaired


This project aims to assist visually impaired users by providing a voice-controlled virtual assistant. The system uses Natural Language Processing (NLP) to understand user commands and respond through text-to-speech technology. It allows users to interact with the system using voice instead of visual interfaces. The assistant performs tasks such as responding to queries and providing basic information through audio feedback. The project focuses on improving accessibility and independence for visually impaired individuals. It was developed using Python and machine learning tools.


Tech Stack: Python, Machine Learning
